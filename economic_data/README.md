# Economic Data
This directory provides a "how to" guide on extracing economic and housing data at the macro level from government and MLS sources.

## Sources
1. [FRED API](https://fred.stlouisfed.org/docs/api/fred/)
2. [Realtor Canada API](https://rapidapi.com/apidojo/api/realty-in-ca1/)
3. [Redfin](https://www.redfin.com/news/data-center/)
4. [Streeteasy](https://streeteasy.com/blog/data-dashboard/)
5. [US Census ACS](https://www.census.gov/data/developers/data-sets/acs-5year.html)
6. [US Housing Data API](https://rapidapi.com/arielherrera/api/us-housing-market-data)
